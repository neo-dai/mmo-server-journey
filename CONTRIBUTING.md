# 贡献指南

感谢你对这个MMO服务器开发学习项目感兴趣！我们欢迎任何形式的贡献。

## 如何贡献

### 报告问题

如果你发现了bug或有改进建议，请通过以下方式提交：

1. 在GitHub上创建一个Issue
2. 详细描述问题或建议
3. 如果可能，提供复现步骤

### 提交代码

1. Fork这个仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个Pull Request

### 代码规范

- 保持代码简洁和可读
- 添加必要的注释（特别是复杂逻辑）
- 遵循项目现有的代码风格
- 确保你的代码可以正常运行

### 项目结构

每个lab项目应该包含：
- `README.md` - 项目说明和实现指南
- `server/` - 服务器端代码
- `client/` - 客户端代码（如果有）

### 学习目标

这个项目旨在帮助学习MMORPG服务器开发，请确保你的贡献：
- 有助于理解MMO服务器开发的核心概念
- 代码结构清晰，便于学习和参考
- 包含必要的文档说明

再次感谢你的贡献！
